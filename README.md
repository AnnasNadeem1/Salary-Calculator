##  Key Excel Functions Used

- `MEDIAN(IF(...))` ‚Äì to calculate filtered median salary
- `COUNTIFS(...)` ‚Äì to count jobs by multiple filters
- `XLOOKUP(...)` ‚Äì to find the top hiring platform
- `SEARCH(...)` ‚Äì for partial match filtering in job types
- `FILTER + SORT` ‚Äì to build dynamic dropdowns and tables
